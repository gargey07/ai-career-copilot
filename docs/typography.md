# Typography

Full rules live in [design-system.md](./design-system.md#3-typography). Quick reference:

Font: **Inter**.

| Role | Size | Weight | Color |
| --- | --- | --- | --- |
| Display (hero) | 44–64px | 800 | text / gradient word |
| H1 page title | 32–36px | 700 | `--text` |
| H2 section | 18–20px | 600 | `--text` |
| Body | 15–16px | 400 | `--text` |
| Label | 13–14px | 500 | `--text-muted` |
| Caption | 12–13px | 400 | `--text-muted` |

Line-height 1.5 body / 1.2 headings. A label is always present above a field — never rely
on the placeholder as the label.
